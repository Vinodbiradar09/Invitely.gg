# Security Policy

## Supported Versions

Only the **latest `main` branch** and active production deployments are supported with security updates.

| Version | Supported |
|---|---|
| `main` | ✅ Yes |
| Older commits | ❌ No |

---

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report privately by emailing: **security@invitely.gg**

Please include:

- A clear description of the vulnerability
- Steps to reproduce (proof of concept if possible)
- Potential impact
- Any relevant logs or screenshots

We aim to:

- Acknowledge your report within **48 hours**
- Provide a fix or mitigation within a reasonable timeframe based on severity

---

## Responsible Disclosure

Please allow us reasonable time to investigate and resolve the issue before any public disclosure. We will credit researchers who follow responsible disclosure practices.

Thank you for helping keep Invitely.gg secure.
